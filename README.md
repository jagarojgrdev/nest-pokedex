# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejectar
```
npm run install
```
3. Instalar Nest CLI
```
npm run install
```
4. Levantar la base de datos
```
docker-compose up -d
```
4. Reconstruir DB con Seed
```
http://localhost:3000/api/v1/seed/
```
